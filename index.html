EXEQ

Banking application:

  • Allows users browsing all bank accounts in one place
  • Provides personalized recommendations to users
  • Analyzes the monthly expenses and is the preferred deposit plan
  • Connects the user with the top asset management firms, provides monitoring of investments and support from professionals

Used bank 3rd-party API:

  • Quovo / Plaid for information about accounts, transactions, personal data, user verification
  • CorePro to create avtomatichkskih deposits
  • TPT, Plaid для создания и верификации брокерского счета
  • CLS for investment management

UI in angular, ionic, cordova. Converted in iOS app.

Backend on NET.MVC 5, Entity Framework

WLS

The project is divided into two parts: the storage of documents and financial services. Programming language: C #. Most of the sites written in ASP.NET Webforms, a small part is written in ASP.NET MVC 5. Most pages do not use the CMS, some of them using CMS Kentico. Database: MS SQL Server, using ADO.NET, using a small part of Entity Framework Code First, there is the use of MongoDB, and text search SOLR engine. The project is integrated with multiple third-party systems, financial services, has a service-oriented architecture, widely used by the WCF. Tools: MS Visual Studio, SQL Server. Source Control: SVN, bug tracking system: Producteev, samopisnaya tracking time system. In the project there is no manager and testers - the programmer is testing the code itself and communicates with the customer.

HR

The system in fact - backend for HR departments of various companies, platform for recruiters. Engine originally built on WebForms, but it is already screwed Kendo, Linq2SQL and WebAPI. Use stateless architecture to facilitate horizontal scaling. At the same time there are a lot of related web sites. For each customer creates a separate site from the template. There are types of projects related to the interaction between the recruiters.

JIRA like Taxco tracker, bitbacket (git) for the source. Appveyor.com for continius delivery; Deployment through https://octopus.com. The methodology in this project - agile, sprints every week (adapts to the customer).

Pick

The application for time management for any size business. Save countless hours of administrative work Eliminate the hassle of pen and paper scheduling Use our powerful administrative software to set up your scheduler Tech. C #, PAT framework, ADO.NET, jQuery / Bootstrap
